Swift 枚举

枚举适用场景：描述一个事物已知的不同状态！

1、枚举的声明与简单使用


/// 枚举是一种数据类型：仅包含有限个可能性
enum Month {
    case January
    case February
    case March
    case April
    case May
    case June
    case July
    case August
    case September
    case October
    case November
    case December
}

/// 使用枚举
let currentMonth = Month.December /// 枚举值不是字符串、整型等；就是 Month 类型

switch currentMonth {
    case .January, .February, .March :
        print("第一季度")
    case .April, .May, .June :
        print("第二季度")
    case .July, .August, .September :
        print("第三季度")
    case .October, .November, .December :
        print("第四季度")
}

枚举中，不同的 case 也可以使用 , 分割

enum Season {
    /// 不同的 case 也可以使用 , 分割
    case Spring, Summer, Autumn, Winter
}

2、枚举的原始值 rawValue

使用 rawValue 将枚举型的一种可能和一个类型的值结合在一起！

在 C、C++ 中枚举型变量的本质还是一个整数，也就是说任何一个枚举型都和一个整型结合！

  • 定义之后，该值不能变化！
/// 显式的将枚举型与整型结合
enum Month : Int {
    case January = 1
    case February = 2
    case March = 3
    case April = 4
    case May = 5
    case June = 6
    case July = 7
    case August = 8
    case September = 9
    case October = 10
    case November = 11
    case December = 12
}

/// 此时 mon 是一个枚举 Month 型
let mon = Month.July

/// 此时 mon 是一个 Int 型
let mon = Month.July.rawValue

/// 将一个整型转为 Month? 型（可选型）
let mon = Month(rawValue: 7) /// 不一定有对应的枚举型，因此返回一个可选型

枚举中，不同的 case 也可以使用 , 分割:

enum Season : Int {
    case Spring = 1, Summer, Autumn, Winter
}

也可以只声明枚举型为 Int, 其 rawValue 由 0 开始依次递增

/// 隐式的将枚举型与整型结合
enum Month : Int {
    case January  /// 默认为 0
    case February /// 默认为 1
    case March
    case April
    case May
    case June
    case July
    case August
    case September
    case October
    case November
    case December
}

在 Swift 中枚举型不仅可以和整型结合，还可以是任何类型结合：

enum Season : String {
    case Spring = "春", Summer = "夏", Autumn , Winter
}
Season.Spring.rawValue /// "春"

/// 注意：使用 String 也有默认值，默认值为枚举名本身
Season.Autumn.rawValue /// "Autumn"

3、枚举的关联值 Associate Value

Associate Value：在 Swift 中对枚举型进行了扩展，支持枚举和一个变量结合！

  • 关联值对枚举型的每一个选项不是必须的，可以视情况在某个枚举使用可选值；
  • Associate Value 不仅可以关联一个值，还可以关联多个值
  • Associate Value 与 rawValue 互斥，如果将枚举型与一个类型结合，则不能再使用 Associate Value；
enum Shape {
    case Square(side : Double) /// 关联一个值
    case Rectangle(width : Double, height : Double) /// 关联两个值
    case Circle(centerX : Double, centerY : Double, radius : Double)
    case Point /// 没有关联值
}

enum Status {
    case Success
    case Error(String) /// Associate Value
}

3.1、枚举型作为函数返回值

在一个请求数据的操作中，有成功和失败两种结果

  • 如果成功，则返回成功码
  • 如果失败，则返回错误信息

这种场景在 Swift 中可以使用 枚举型来实现：

func request(success:Bool) -> Status { /// 返回一个枚举型
    if success {
        return .Success
    } else {
        return .Error("失败的具体原因")
    }
}

let result = request(success: false)
switch result {
case .Success: /// 对成功信息无需关注，直接返回状态
    print("请求成功")
case let .Error(errorMsg) : /// 需要知道失败的具体原因，因此对 Error 解码
    print("错误信息 : \(errorMsg)")
}

3.2、枚举型作为函数参数值

传入一个枚举型 Shape ，计算该形状的面积：

enum Shape {
    case Square(side : Double)
    case Rectangle(width : Double, height : Double)
    case Circle(centerX : Double, centerY : Double, radius : Double)
    case Point
}

func area(shape : Shape) -> Double {
    switch shape {
        case let .Square(side) :
            return side * side
        case let .Rectangle(width, height) :
            return width * height
        case let .Circle(_, _, radius) : /// _ 忽略没有使用的参数
            return Double.pi * radius * radius
        case .Point :
            return 0
    }
}

/// 声明一个圆
let cir = Shape.Circle(centerX: 0, centerY: 0, radius: 10)
/// 计算圆的面积
area(shape: cir)

4、枚举中定义方法

不同于其它语言，在 Swift 中也可以在枚举体中定义一个方法！

enum Shape {
    case Square(side : Double)
    case Rectangle(width : Double, height : Double)
    case Circle(centerX : Double, centerY : Double, radius : Double)
    case Point

    func area() -> Double {
        switch self {
            case let .Square(side) :
                return side * side
            case let .Rectangle(width, height) :
                return width * height
            case let .Circle(_, _, radius) : /// _ 忽略没有使用的参数
                return Double.pi * radius * radius
            case .Point :
                return 0
        }
    }
}

/// 声明一个圆
let cir = Shape.Circle(centerX: 0, centerY: 0, radius: 10)
/// 计算圆的面积
cir.area()

5、可选型的本质就是枚举

可选型既可以是一个具体的类型值，也可以是 nil！

/// 可选型的声明
var greeting : String?
var greeting : Optional<String>

/// 可选型的赋值
greeting = Optional.some("Hello")

Optional 实质是一个枚举型：some 是枚举类型的一个选择，关联一些值！

/// 使用枚举型对 Optional 进行了一层包装
enum Optional<Wrapped>

对可选型赋空操作：

greeting = .none /// 枚举 -- 没有
greeting = nil /// 直接赋空

对可选型解包:

/// 使用 Switch 解包
switch greeting {
case .none :
    print("none")
case let .some(value) :
    print(value)
}

/// 使用 if-else 解包
if let greeting = greeting {
    print(greeting)
} else {
    print("none")
}

6、枚举之递归

递归型枚举：枚举值关联自身的枚举！

indirect enum ArithmeticalExpression {
    case Number(Int)
    case Add(ArithmeticalExpression, ArithmeticalExpression) /// 关联值仍是该枚举型
    case Multi(ArithmeticalExpression, ArithmeticalExpression)
}

注意：编译器并不知道为递归型枚举开辟多大的内存空间，可能无限递归下去！因此需要使用关键字 indirect 显式声明递归型枚举，此时编译器内部会使用指针指向该枚举，具体的内存处理由编译器完成！

/// 递归枚举型的声明
let five = ArithmeticalExpression.Number(5)
let four = ArithmeticalExpression.Number(4)
let sum = ArithmeticalExpression.Add(five, four)

/// 解包计算递归型枚举
func evaluate(expression : ArithmeticalExpression) -> Int {
    switch expression {
    case let .Number(value) :
        return value
    case let .Add(left, right) :
        return evaluate(expression: left) + evaluate(expression: right)
    case let .Multi(left, right) :
        return evaluate(expression: left) * evaluate(expression: right)
    }
}

/// 解析递归型枚举
evaluate(expression: sum) /// 9

7、枚举型属于值类型

枚举型是一个值类型，编译器在处理枚举型的时候，当做一个值处理！

值类型的显著特点就是 赋值即是拷贝！

enum Season : String {
    case Spring = "春", Summer = "夏", Autumn , Winter
}

var sea1 = Season.Spring
var sea2 = sea1 /// 将 sea1 赋值给 sea2 ，内部做了拷贝操作

/// 验证：改变 sea2 的值，sea1 并没有跟着改变 =》 sea1 与 sea2 是两块内存空间
sea2 = .Summer
sea1 == sea2 /// false

Swift 中，结构体属于值类型，容器类 Array 、Dictionary 、 Set, 字符串String 本质都是结构类型，因此它们都是值类型！
