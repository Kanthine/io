Swift 结构体

1、结构体的声明与使用

定义一个结构体：

struct Point {
    var x : Float
    var y : Float
    let z : Float
}

注意：只有结构体的成员类型是变量，声明的结构体也是变量；才可以修改该成员值！否则常量值不能修改。

/// 声明一个结构体常量，并使用默认构造函数来初始化
let point1 = Point(x: 10, y: 10, z: 10) /// 默认构造函数
point1.x /// 访问结构体中的某一成员
point1.y = 200 /// 不能修改：Cannot assign to property: 'point1' is a 'let' constant

/// 声明一个结构体常量并初始化
let point2 = Point(x: 10, y: 10, z: 10)
point2.y = 200 
point2.z = 300 /// 不能修改：Cannot assign to property: 'z' is a 'let' constant

2、结构体的构造函数

2.1、常量成员并赋初值

在结构体成员中，如果成员有初始值，设置为常量 let 意义并不大！

struct Point {
    var x : Float = 0
    var y : Float = 0
    let z : Float = 0 /// 为一个常量成员赋予初值
}

/// 如果结构体成员有默认值，构造函数的参数列表可以重载
let point = Point() 
let point = Point(x: 10, y: 10)
let point = Point(x: 10)
let point = Point(y: 10)

/// Error：此时不论是构造函数、还是访问成员，都不能再次修改
let point = Point(x: 10, y: 10, z = 10)

如果需要一个常量成员，最好在初始化的时候赋值：

struct Point {
    var x : Float = 0
    var y : Float = 0
    let z : Float /// 常量成员
}

let poitn = Point(x: 10, y: 12, z: 10)
let point = Point(y: 10, z: 10)

2.2、自定义构造函数

可以在结构体中自定义一个构造函数：

  • 注意：此时系统提供的默认构造函数已消失，无法使用！

Swift 是一门安全的语言：

  • 结构体成员要么指定初值，要么处于没有初始化的状态；
  • 没有初始化的状态，就不能使用；
  • 建议结构体或者类，有一个包含全参数的构造函数； 
struct Point {
    var x : Float = 0
    var y : Float = 0

    /// 没有默认值（没有就是没有，是一种状态）
    let z : Float

    init(string: String) {
        let arr = string.components(separatedBy: ",")
        x = Float(arr[0]) ?? 0
        y = Float(arr[1]) ?? 0
        z = Float(arr[2]) ?? 0
    }

    /// 系统提供的默认构造函数已消失
    /// 如果想使用，需要开发者去声明
    init(_ x : Float, _ y : Float, _ z : Float) {
        self.x = x
        self.y = y
        self.z = z /// 变量名产生歧义的场景，需要使用 self 来区分不同的变量
    }
}

var point = Point(string: "10,20,30")
var point = Point(10, 10, 10)

在上述构构造一个结构体时，可能传递一个非法的字符串，这时程序就会报错！Swift 是一门安全的语言，在执行逻辑之前，需要保证传入的字符串是合法的，否则返回 nil！

struct Point {
    var x : Float = 0
    var y : Float = 0    
    let z : Float

    /// Fail-Initializer
    init?(string: String) {
        let arr = string.components(separatedBy: ",")

        /// 使用 guard 保证传递的字符串合法，否则返回 nil
        guard arr.count == 3, 
              let x = Float(arr[0]), 
              let y = Float(arr[1]), 
              let z = Float(arr[2])
        else {
            return nil
        }

        self.x = x
        self.y = y
        self.z = z
    }
}

3、在结构体中定义函数

struct Point {
    var x : Float = 0
    var y : Float = 0    
    let z : Float

    init(_ x : Float, _ y : Float, _ z : Float) {
        self.x = x
        self.y = y
        self.z = z
    }

    func isEqualTo(point:Point) -> Bool {
        return self.x == point.x && self.y == point.y && self.z == point.z
    }
}

let point1 = Point(10, 10, 10)
let point2 = Point(20, 20, 20)
point1.isEqualTo(point2) /// false

4、结构体是一个值类型

结构体是一个值类型，编译器在处理结构体的时候，当做一个值处理！

值类型的显著特点就是 赋值即是拷贝！

var point1 = Point(10, 10, 10) 
var point2 = point1  /// 将 p1 赋值给 p2 ，内部做了拷贝操作

/// 验证：改变 p2 的成员值，p1 并没有跟着改变 =》 p1 与 p2 是两块内存空间
point2.x += 1
point2.x == point1.x /// false

4.1、Swift 中结构体的延伸

在 Swift 中，很多数据类型本质都是结构体类型，这些类型都属于值类型：

  • 容器类 Array 、Dictionary 、 Set；
  • 字符串 String
  • 基本数据类型 Int 、Float 、 Double、 Bool 等；
